Git コマンド一覧

ファイル一覧
git ls-files

ファイル削除
git rm ファイル名

初期化
git init

追加
git add ファイル名

コミット
git commit -m ‘コメント'

ステージングのキャンセル
git rm —cached ファイル

リモートリポジトリプッシュ
git push -u origin origin

リモート削除
git push --delete origin master
